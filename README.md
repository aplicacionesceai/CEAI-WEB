# Página Web CEAI - Área de Innovación y Competitividad

## Descripción
Página web institucional de divulgación para el área de Innovación y Competitividad del CEAI, Regional Valle del SENA.

## Requerimientos
- Home con noticias actualizables
- Catálogo de semilleros
- Proyectos por semillero
- Formulario de contacto
- Responsive design
- Identidad visual SENA

## Tecnologías usadas
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Git + GitHub
